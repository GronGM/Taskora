## Правила

- Используйте только тестовые данные.
- Не загружайте реальные документы.
- Не вводите паспортные данные.
- Не вводите банковские карты.
- Реальные платежи и выплаты не подключены.
- Все оплаты работают как заглушка.

## С чего начать

- Откройте `/beta-testing`.
- Выберите роль.
- Пройдите чек-лист.
- Ошибки отправляйте через `/beta-feedback/create`.
